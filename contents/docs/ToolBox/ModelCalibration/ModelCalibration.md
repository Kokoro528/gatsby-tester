# 模型标定

对于物理系统建模与仿真，一般情况下其中的组件（部件或子系统）会包含很多参数，这些参数对应实际的环境参数、运行工况等，模型标定即是通过改变一些模型参数使得仿真结果变量与测量数据达到吻合。
